ApplyPilot ‚Äî AI Resume + Job Application Tracker (Enterprise SaaS)

PROJECT IDENTITY
--------------------------------------------------------------------------------
Name: ApplyPilot
Type: SaaS Web Application
Core Purpose: AI-powered resume tailoring + application tracking + reminders
Target Users: Students, job seekers, internship seekers, international applicants
Long-term Goal: Scale from 10,000 ‚Üí 100,000 users without performance issues
Current Status: Phase 1 (Foundation) ‚Äî Core architecture implemented

EXECUTIVE SUMMARY
--------------------------------------------------------------------------------
ApplyPilot is an AI-powered platform that helps users:
- Manage and store resumes securely with version control
- Tailor resumes to job descriptions using AI matching
- Generate professional content (cover letters, follow-ups, interview prep)
- Track applications in a visual pipeline (Saved ‚Üí Applied ‚Üí Interview ‚Üí Offer ‚Üí Rejected)
- Receive smart reminders and notifications (in-app + email)
- Maintain enterprise-grade security with verification workflows

MVP CORE FEATURES
--------------------------------------------------------------------------------
1) Authentication & User Management
        - User registration with email verification
        - Login/Logout with JWT access + refresh tokens
        - Password hashing (bcrypt)
        - Forgot password / reset password workflow
        - All protected resources require valid authentication

2) Dashboard (Primary UI Entry Point)
        - Sidebar navigation and unified page layout
        - Header with search + New Application action
        - Stat cards (weekly applications, interviews, offers, AI credits)
        - Pipeline breakdown by status
        - Upcoming follow-ups list
        - Recent activity timeline
        - Quick actions (Tailor Resume, Cover Letter, ATS Scan, Add Reminder)
        - Design consistency across all pages

3) Application Tracking
        - Job record: company, job title, link, description, status, follow-up date, notes
        - Status pipeline: Saved ‚Üí Applied ‚Üí Interview ‚Üí Offer ‚Üí Rejected
        - CRUD operations + timeline history
        - Search/filter on pipeline
        - Note: platform does NOT auto-apply to jobs

4) Resume Management
        - Upload resume (PDF/DOCX)
        - Extract text automatically
        - Store metadata in Postgres, file in object storage
        - Master resume + tailored versions per job

5) AI Engine
        - Tailor resume to job description
        - Generate cover letter
        - ATS keyword checklist
        - Suggest missing skills/keywords
        - Optional interview prep suggestions
        - AI usage tracking per user

6) Notifications & Reminders
        - In-app notifications with read/unread state
        - Reminder types: follow-ups, status updates, AI completion
        - Email notifications via SendGrid

SECURITY REQUIREMENTS
--------------------------------------------------------------------------------
Authentication
- bcrypt password hashing
- JWT access (short-lived) + refresh tokens (long-lived)
- Refresh token rotation (future)

Authorization
- Validate user ownership via user_id
- RBAC foundation for future

Email Security
- Verification + reset tokens are single-use and expire

API Security
- Rate limiting (auth + AI endpoints)
- File upload validation (size/type/malware scan)
- CORS restricted to trusted domains
- ORM-based SQL injection prevention
